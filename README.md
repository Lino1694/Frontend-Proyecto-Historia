## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`


## Ollama Service

**Prerequisites: download and install ollama in your device**

1. Open cmd
2. Install models:
`ollama pull gemma2:2b`
`ollama pull llama3.2:3b`
`ollama pull phi3:mini`
3. Check installation
`ollama list`
